What a workflow
looks like.

meowlands-content-pipeline · n8n workflow
// Trigger: every day at 09:00 UTC
01 → apify.run("trending-topics-scraper")
// Returns: ["AI tools", "automation", "indie dev"]

02 → claude.messages.create({
    system: "You are @MeowlandsNFT. Edgy, direct, no fluff.",
    prompt: "Write 3 posts about: {{trending_topics}}"
})

03 → stableDiffusion.generate("{{post_visual_prompt}}")
// Returns: image matching post context

04 → twitter.post({ text, image, scheduledAt: "peak_hours" })

// Done. Loop runs again tomorrow.
